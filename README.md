# Chargepoint

A gem to wrap the ChargePoint network JSON APIs

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'chargepoint'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install chargepoint
